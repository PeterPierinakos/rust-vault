RustVault is a secure way to store your plaintext notes on the cloud. Completely free and open source. Licensed under GPL v3.0

QNA

What language was this website written in?

The backend of this website was written completely in Rust, so expect blazing fast speeds if you self host. Front-end is written in vanilla JavaScript because there is no point in using React or WASM for a static HTML front-end.

Q. What metadata is collected?

Absolutely none. Only username and password is stored on the database for authentication.

Q. License?

Licensed under the open source GPL v3.0 license. This project is completely open source and distributable as long as you keep the license.

Q. How many accounts and plaintext files can I store?

Unlimited.
